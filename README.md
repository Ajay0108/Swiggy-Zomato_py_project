🚀 Swiggy & Zomato Delivery Analysis

This project performs Exploratory Data Analysis (EDA) on a large-scale food delivery dataset (450K+ records) to uncover operational insights and improve delivery performance.

🔍 Key Objectives
Analyze delivery efficiency and identify slow-performing riders
Detect peak order hours for surge pricing optimization
Study distance vs cancellation correlation
⚙️ Tech Stack
Python (Pandas, NumPy)
Data Visualization (Seaborn, Matplotlib)
📊 Key Features
Data cleaning & preprocessing of real-world delivery data
Feature engineering (delivery duration, time buckets, etc.)
Time-based analysis (hour, weekday, monthly trends)
Rider performance evaluation
Business-driven insights for operations & marketing
📁 Dataset
Swiggy & Zomato order-level data (Kaggle)
